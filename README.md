# EgovFramework Project - Client
전자정부프레임워크 + JSP를 사용한 클라이언트 페이지
<br/>
<br/>

## 개발 환경
 - JDK "1.8"
 - IDE "Eclipse"
 - Framework "EgovFramework 3.7"
 - Database "Oracle DB 11xe"
 - ORM "Ibatis"

## 구현 기능
### 1. 회원 기능
 - 로그인
 - 회원가입
 - 회원 정보 수정

### 2. 게시판
 - 게시글 등록
 - 게시글 수정
 - 게시글 삭제
 - 게시글 조회
 - 검색
 - 페이징
